# Easy install

```bash
npm install tumblr-builder
```

::: warning Tumblr API usage
`tumblr-builder` use Tumblr [API v2](https://www.tumblr.com/docs/en/api/v2) and need a valid `apiKey`. More information in the guide.
:::
